For Loops

Para começar vamos usar um codigo simples:

#include <stdio.h>
#include <conio.h>
#include <stdlib.h>
#include <locale.h>
int main(){
    setlocale(LC_ALL, "portuguese");
    int vec[10];
    printf("Digite 10 Numeros:\n");
    for(int i = 0; i < 10; i++){
        scanf("%i", &vec[i]);
    }
    getch();
}

Tudo o que o programa deve fazer e ler 10 números do usuario e guarda-los em sua posição correta em vec[10]. Porém o programa poderia ser escrito assim que funcionaria do mesmo jeito:

int vec[10];
printf("Digite 10 Numeros:\n");
    scanf("%i", &vec[0]);
    scanf("%i", &vec[1]);
    scanf("%i", &vec[2]);
    scanf("%i", &vec[3]);
    scanf("%i", &vec[4]);
    scanf("%i", &vec[5]);
    scanf("%i", &vec[6]);
    scanf("%i", &vec[7]);
    scanf("%i", &vec[8]);
    scanf("%i", &vec[9]);

O que seria uma péssima ideia se tivermos que repetir o mesmo porém com um vetor de 100 posições. Para evitar este tipo de coisa foram introduzidos os loops, que são estruturas de repetição programaveis que podem repetir um bloco de codigo seguindo uma determinada logica, esta logica é escrita entre os ( ; ; ) da função:

 for ( int i = 0 ; i < 10 ; i++) {
    scanf("%i", &vec[i]);
}

Este loop no caso vai rodar o codigo dentro dos { } 10 vezes, mais poderia ser quantas vezes forem nessesarias.
Os for loop sempre servem para repetir um trecho de codigo, mudando apenas uma variavel, seja para mudar a posição de um vetor, ou para fazer contas tipo f(x).
O codigo em si não é obrigado a usar a variavel de contagem, por exemplo:

 for ( int i = 0; i < 50 ; i++) {
    printf("hello word);
}

Este loop irá imprimir hello world 10 vezes, e não está usando i nenhuma vez, Já este usa o i para varias coias.

 for ( int i = 0; i < 50 ; i++) {
    printf("%i",i + 1);
    printf("%i",i * 2);
    printf("%i",i / i + 2);
    printf("%i",vec[i] / mat[0][i]);
}

Variavel de contagem


 for ( int i = 0; i < 10 ; i++) {
    scanf("%i", &vec[i]);
}

A primeira parte do for loop e a variavel de contagem, e sobre ela que o for loop ira agir e aplicar seu efeito ao bloco de codigo. A variavel de contagem pode ter qualquer nome (dentro das regras) e é comum inicializarmos esta variavel a 0, porém é possivel inicializa-la com qualquer valor, caso voccê queira um loop que va de 100 a 200 por exemplo.
Dentro do for loop a variavel esta mudando o indice do vetor para o valor do usuario ir para uma posição diferente, Nós podemos usa-la quantas vezes forem nessesarias dentro do for loop, ela não serve apenas para trocar indices de vetores. Ela é apenas um numero comum, como deveria ser aqui.

Condição


 for ( int i = 0; i < 10 ; i++) {
    scanf("%i", &vec[i]);
}

A segunda parte do for loop é uma condição, ela é executada uma vez por iteração do loop, enquanto ela for true, o loop continuará, e irá parar quando for false, por isto é bom tomar cuidado para não criar loops infinitos. Eis alguns exemplos.

for(int i = 0; true; i++){
    //Este loop nunca ira parar pois a conição sempre  ser true
    printf("%i", i);
}

for(int i = 0; i <= 10; i--){
    //Este nunca ira parar pois a condição espera i ser menor ou igual
    //a 10, o que nunca irá acontecer pois o loop anda para trás (note o i--)
    printf("%i", i);
    //irá imprimir
    //0,-1,-2,-3...
}

for(int i = 0; i < 100; i++){
    //Este por si só funciona, porém o codigo tem um processamento
    //que faz o loop voltar para 0 
    if(i >= 99){
        i = 0;
    }   
    printf("%i", i);
    //Isto pode acontecer por causa de confusões com nomes de variaveis
    //Ou por processamentos mal programados
}

for(int i = 0; !(i == 50); i+=3){
    //A ideia deste loop seria parar exatamente em 50
    //porem o incremento está de 3 em 3, logo i nunca será 50
    printf("%i", i);
    //irá imprimir
    //0,3,6,9,12,15,18,21,24,27,30,33,36,39,42,45,48,51,54...
}

Loops infinitos não são tão perigosos assim em programas simples em c++, porém esses loops em programas complexos podem acabar usando muitos recursos do computador, em outras linguagems como javascript os loops infinitos travam completamente o seu navegador e deixam o computador bastante lento. Incremento


 for ( int i = 0; i < 10 ; i++) {
    printf("%i", i);
}

O incremento é uma das partes mais importantes do loop, é ela que diz como que a variavel de contagem deve se portar no decorrer do programa e é ela que faz com que a condição uma hora chegue em false.
E até que facil entender seu funcionamento, desde que a condição do loop uma hora chegue em false.

for(int i = 0; i <= 1000; i+=2){
    //Um loop que conta em incrementos de 2
    printf("%i", i);
}
for(int i = 0; i >= 1000; i-=4 ){
    //Um loop que conta em decrementos de 4
    printf("%i", i);
    //Note a condição i >= 100 (enquato i for menor ou iqual a 1000)
}
for(int i = 0; i <= 1000; i*=4 ){
    //Um loop que a cada incremento multiplica i por 4 
    printf("%i", i);
    //Note a condição i >= 100 (enquato i for menor ou iqual a 1000)
}
for(int i = 0; i <= 1000; i % 2 == 0 ? i+=6 : i+=3 ){
    /*Um loop que toda vez checa se i é par
    *se for ira somar + 6 a i
    *se não ira somar + 3 a i*/
    printf("%i", i);
}
